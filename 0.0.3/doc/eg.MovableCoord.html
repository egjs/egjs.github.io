See:
  • Hammerjs http://hammerjs.github.io
  • There is usability issue due to default CSS properties (http://hammerjs.github.io/jsdoc/Hammer.defaults.cssProps.html) settings from Hammerjs. movableCoord removes that settings to fix. Hammerjs의 기본 CSS 속성(http://hammerjs.github.io/jsdoc/Hammer.defaults.cssProps.html) 으로 인해 사용성 이슈가 있다. 따라서, movableCoord는 hammerjs의 기본 CSS 속성을 제거하였다.
  • Easing Functions Cheat Sheet http://easings.net/
  • To use other easing functions, import jQuery easing plugin(http://gsgd.co.uk/sandbox/jquery/easing/) or jQuery UI easing.(https://jqueryui.com/easing/)다른 easing 함수를 사용하고 싶다면, jQuery easing plugin(http://gsgd.co.uk/sandbox/jquery/easing/)이나, jQuery UI easing(https://jqueryui.com/easing/) 라이브러리를 삽입해야 한다.
